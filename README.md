* 数组
    - 二分
    - 双指针
    - DP
    - Hash结构辅助
    - 滑动窗口
    - 前缀和
    - 位运算
    - 递归
* 链表
    - 快慢指针
    - 递归
* 字符串
    - Hash结构辅助
    - 位图
    - 双指针窗口
    - DP
    - KMP算法
    - Manacher算法
    - Rabin-Karp算法
* 树
    - 递归
    - 栈和队列
